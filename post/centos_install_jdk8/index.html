一.百度搜索jdk8

二.进入oracle官网下载页面，鼠标放在Sign in 上，点击 Sign in to my Account

三.有账号的输入账号后点击登录，没有账号的创建一个账号后登录

四.登录后点击Accept License Agreement, 然后点击jdk-8u231-linux-x64.rpm后下载jdk8

五.下载结束后将下载的jdk包上传到你要安装的虚拟机

# 执行ll查看你上传的文件
总用量 174404
-rw-r--r--. 1 root  root  178589686 12月 30 23:19 jdk-8u231-linux-x64.rpm

六.安装jdk8：-i：显示套件的相关信息，-v：显示指令执行过程；-h或–hash：套件安装时列出标记

rpm -ivh jdk-8u231-linux-x64.rpm

七.用“java -version”的命令查看java的环境变量是否生效,出现下面的内容就说明java的jdk已经部署完成

java version "1.8.0_231"
Java(TM) SE Runtime Environment (build 1.8.0_231-b11)
Java HotSpot(TM) 64-Bit Server VM (build 25.231-b11, mixed mode)
linux
