HTML元素分类

发表于 2016-09-02

Table of Contents

在HTML中，HTML标签元素大体被分为三种不同的类型：内联元素(又叫行内元素)，块状元素，内联块状元素。

一.常用的内联元素有： 

199<a>、<span>、<br />、<i>、<em>、 <strong>、<label>、<q>、<var>、<cite>、<code>
内联元素的特点：
1.和其他内联元素都显示在同一行
2.元素的宽度、高度、顶部边距和底部边距不可设置
3.元素的宽度就是它包含的文字或图片的宽度，不可改变
可通过CSS代码display:inline将元素设置为内联元素: 
199    div {
200        display: inline;
201    }

二.常用的块状元素有： 
199<div>、<p>、<h1>…<h6 />、<ul>、<ol>、 <table>、<address>、<blockquote>、<form>
块状元素的特点：
1.每个块状元素都从新的一行开始，并切其后的元素也另起一行(一个块状元素独占一行)
2.元素的宽度、高度、顶部边距和底部边距都可设置
3.元素的宽度在不设置的情况下，是他本身父容器的100%
可通过CSS代码display:block将元素设置为块状元素： 
199    div {
200        display: block;
201    }

三.常用的内联块状元素有： 
199<img>、<input>
内联块状元素的特点：
1. 和其他内联元素或者内联块状元素都显示在同一行
2. 元素的宽度、高度、顶部边距和底部边距都可设置
可通过CSS代码display:inline-block将元素设置为内联块状元素： 
199    div {
200        display: inline-block;
201    }
